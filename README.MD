# Autenticação 2fa Frontend

#### Crie um arquivo .env

**Exemplo:**

```
VITE_URL_API = http://localhost:9090

```

## :muscle: Execução

#### Execute

```bash
npm i
```

#### Starta a aplicação

```bash
npm run dev
```
